# **App Name**: ColserDev Pro Site Redesign

## Core Features:

- Responsive Homepage Display: Replicate the professional, modern homepage layout shown in the provided image, ensuring full responsiveness across all device sizes, including the main hero section and partner logos.
- Dynamic Navigation Bar: Implement the header navigation from the design image with interactive links and a clear 'Get a Quote' call-to-action button, ensuring smooth user experience.
- Modular Services & Projects Showcase: Develop a section featuring interactive 'cards' to highlight services and past projects, as suggested by the 'Projects' navigation link and the general aesthetic of the provided image.
- Quote & Contact Form Interface: Create front-end forms for 'Get a Quote' and 'Start Your Project' CTAs, enabling users to submit inquiries. This implies an 'other' backend integration for submissions, though the MVP focuses on the UI.
- AI-Powered Content Idea Tool: An internal generative AI tool for ColserDev team members to quickly brainstorm and draft initial content ideas for service descriptions or blog posts, ensuring messaging consistency and accelerating content creation.

## Style Guidelines:

- The primary interactive color is a bright, modern blue (#1F8EFC), chosen to reflect innovation and professionalism, used for calls-to-action and key text highlights.
- The main background color is a subtle, almost off-white with a hint of blue (#ECF4FC), providing a clean, spacious canvas that complements the primary blue and enhances readability.
- An accent color, a soft, lighter teal (#98E0D8), will be used for secondary interactive elements or subtle design details, offering an analogous yet distinct contrast.
- The entire site will use 'Inter', a modern sans-serif font, for both headlines and body text, providing excellent readability and a contemporary, professional appearance.
- Utilize a consistent set of clean, linear vector icons that align with the brand's tech-focused and modern image, as seen in the ColserDevPro logo with the stylized rocket.
- A responsive, grid-based layout with ample whitespace to maintain a clean aesthetic. Key sections will be prominently center-aligned, with content structured to be easily digestible.
- Subtle, smooth animations for hover states on interactive elements and card transitions to add a dynamic and polished feel without being distracting, enhancing the user experience.
