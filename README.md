## 🚀 Pasos para el Despliegue en Firebase App Hosting

Sigue estos pasos para poner tu sitio en vivo con tu dominio `colserdev.com` usando tu repositorio actual:

### 1. Subir el Código a GitHub
Asegúrate de que todo el código actual esté subido a tu repositorio:
`https://github.com/Godie84/colserdev_page.git`

### 2. Configurar en Firebase Console
1. Ve a [Firebase Console](https://console.firebase.google.com/).
2. Selecciona tu proyecto (o crea uno nuevo).
3. En el menú lateral izquierdo, ve a **Build** > **App Hosting**.
4. Haz clic en **Get Started** o **Comenzar**.
5. Conecta tu cuenta de GitHub y selecciona el repositorio `Godie84/colserdev_page`.
6. Configura los ajustes de despliegue:
   - **Branch:** `main` (o la rama que estés usando).
   - **Root directory:** `/` (raíz).
7. Haz clic en **Finish and Deploy**. Firebase detectará automáticamente que es un proyecto de Next.js.

### 3. Vincular Dominio Personalizado
Una vez que el despliegue inicial termine:
1. En la pestaña de App Hosting, ve a **Settings** (Configuración) de tu backend creado.
2. En la sección **Custom Domains**, haz clic en **Add Domain**.
3. Ingresa `www.colserdev.com`.
4. Firebase te proporcionará los registros DNS (A, AAAA o CNAME). Cópialos y pégalos en la configuración DNS de tu proveedor de dominio.
