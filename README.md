## 🚀 Pasos para el Despliegue en Firebase App Hosting

Sigue estos pasos para poner tu sitio en vivo con tu dominio `colserdev.com`:

### 1. Preparar el Repositorio
Asegúrate de que este código esté en un repositorio de **GitHub**. Firebase App Hosting utiliza una conexión directa con GitHub para realizar despliegues automáticos (CI/CD).

### 2. Configurar en Firebase Console
1. Ve a [Firebase Console](https://console.firebase.google.com/).
2. Selecciona tu proyecto.
3. En el menú lateral izquierdo, ve a **Build** > **App Hosting**.
4. Haz clic en **Get Started** o **Comenzar**.
5. Conecta tu cuenta de GitHub y selecciona el repositorio de ColserDev.
6. Configura los ajustes de despliegue:
   - **Branch:** `main` (o la rama principal que uses).
   - **Root directory:** `/` (raíz).
7. Haz clic en **Finish and Deploy**.

### 3. Vincular Dominio Personalizado
Una vez que el despliegue inicial termine:
1. En la pestaña de App Hosting, ve a **Settings** (Configuración) de tu backend.
2. En **Custom Domains**, haz clic en **Add Domain**.
3. Ingresa `www.colserdev.com`.
4. Firebase te proporcionará registros DNS (A, AAAA o CNAME). Copia estos valores y pégalos en la configuración DNS de tu proveedor de dominio (donde compraste colserdev.com).
